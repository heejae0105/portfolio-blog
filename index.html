화려한 코드나 복잡한 알고리즘보다 더 중요한 것은 '이 기술이 왜 필요한가'에 대한 물음이라고 믿습니다.

일상 속의 작은 불편함이나 사회적 문제들을 발견하면, 그것을 방치하기보다 제가 가진 기술로 어떻게 해결할 수 있을지 고민하는 과정을 즐깁니다. 단순히 작동하는 프로그램을 넘어, 사람에게 닿는 가치를 만드는 엔지니어가 되기 위해 공부하고 기록합니다.

Selected Projects
2025 IoT / Embedded

다인 탑승 감지 시스템

전동 킥보드의 2인 이상 탑승 사고를 예방하기 위해 FSR 압력 센서와 초음파 센서를 융합(Sensor Fusion)하여 제작한 오탐지 방지 시스템입니다.

Problem & Solution

단순 무게 감지만으로는 물건 적재와 구분이 어려운 한계를 발견했습니다. 이에 '압력 분포'와 '탑승 공간 밀집도'라는 두 가지 변수를 교차 검증하는 알고리즘을 설계하여 정확도를 높였습니다.

Key Learning

  • 하드웨어 센서 데이터의 노이즈 처리 및 필터링 경험
  • 사용자의 실제 행동 패턴을 고려한 예외 상황 처리(Edge Case) 로직 설계
  • 공학적 설계가 실제 사회 안전 문제에 기여하는 과정 체감
2025 Blockchain / Network

P2P 탈중앙 메신저 프로토타입

중앙 서버 마비 사태에서도 끊기지 않는 소통을 위해, Python Socket 통신과 블록체인 구조를 활용해 구현한 서버리스 메신저입니다.

Focus

편리함 뒤에 가려진 중앙 집중형 구조(SPOF)의 취약점을 분석하고, '신뢰의 프로토콜'로서 블록체인이 어떻게 데이터 주권을 개인에게 돌려줄 수 있는지 기술적으로 검증했습니다.

  • Python Socket을 활용한 Peer-to-Peer 통신 구현
  • 메시지 블록 해시 연결을 통한 데이터 무결성 검증
2025 AI / NLP / Web

DealChecker: AI 계약서 분석

법률 지식이 부족한 일반인을 위해, 자연어 처리 모델을 파인튜닝하여 계약서 내 독소 조항을 자동으로 탐지하고 시각화하는 웹 서비스입니다.

Impact

기술이 사회적 약자의 정보 비대칭 문제를 해결하는 도구가 될 수 있음을 확인했습니다. 복잡한 법률 용어를 AI가 분석하여 직관적인 위험도로 변환해 제공합니다.
